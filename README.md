# Mini Game Board

A collection of lightweight, web-based mini-games built with TypeScript, Pug, and SCSS.

## Project Structure

- `src/`: Source code for all games and the main index.
  - `index.pug`: The landing page template.
  - `script.ts`: Logic for the game index/selection.
  - `dark-maze/`: Source for the "Dark Maze" game.
  - `tank-shot/`: Source for the "Tank Shot" game.
- `dist/`: Compiled output (generated by the build process).
- `build.js`: Custom build script using `@nmhillusion/n2ngin-bull-engine`.

## Build System

The project uses a custom build engine that handles:
- **TypeScript**: Compiled to ES Modules.
- **Pug**: Compiled to HTML.
- **SCSS**: Compiled to CSS.
- **Asset Rewriting**: Automatically handles import path adjustments for the browser.

### Commands
- `npm run build`: Executes `node build.js` to compile the project.

## Games

### 1. Dark Maze
A grid-based survival maze game with:
- Fog of War (Revealed by "touches" or movement).
- Procedural maze generation.
- Monsters (Slow/Fast) and Bombs.
- Level progression system.
- Responsive mobile controls.

### 2. Tank Shot
A dark, intense tactical tank combat game featuring:
- Class-based Bot AI (Scout, Heavy, Tactician).
- Procedural "Force Shield" effects and spawning protection.
- Destructible and permanent (steel) environment.
- Power-ups including penetrating bullets and health boosters.
- Action-packed industrial chiptune soundtrack.
